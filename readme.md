Q: 如何运行？

A: 在ubuntu环境中，进入目录后，执行make run可以看到在虚拟机中运行的效果。

Q:如何查看bin文件？

A: xxd -a -u -g 1 xxx.bin或者hexdump -C xxx.bin

Q:如何查看obj文件？

A: objdump -M intel -d *.o;  -M intel使得结果按intel asm格式显示，否则是AT&T格式。
